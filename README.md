# Cube Summation

The next project present a posible solution for cube summation problem, using Spring framework, Rest web services, Html templates and file interchange from GUI.

## Development Installation

1. Clone repository
2. Open with Eclipse IDE
3. Right click on the project and select Maven, and later update project.
4. Right click on the project and select Run as Java Application, at this step you must select the main class call "CubeSummationApplication".
5. Open http://localhost:8080/

## Production Intallation

1. If you don´t have maven, please install it: https://maven.apache.org/install.html
2. With console command cd in cube-summation and run ./mvnw spring-boot:run
3. The last command downloads mvn dependencies and also runs server.
4. Open http://localhost:8080/
